# Código de Conduta do Colaborador

Este projeto adota o [Contributor Covenant](https://www.contributor-covenant.org/) como guia de conduta.

## Nossos Padrões

- Mostrar empatia, respeito e cordialidade.
- Aceitar críticas construtivas com maturidade.
- Colaborar buscando o melhor resultado técnico.

## Responsabilidades da Manutenção

A equipe responsável pelo projeto pode remover ou bloquear qualquer contribuição ou colaborador que desrespeite este código.

## Escopo

Aplica-se a todos os espaços do projeto, incluindo repositórios, issues, Pull Requests e canais de chat associados.

## Cumprimento

Para reportar violações, entre em contato com os mantenedores através dos canais definidos no `README.md`.
